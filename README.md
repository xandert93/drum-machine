# drum-machine
Mobile-first React application. Built with:

- Stateful Class Components and Stateless Functional Components
- the Class Fields syntax
- the Context API
- Refs
- Error boundaries
- Lifecycle Methods
- HTML Media API
- SASS + BEM
- CSS Flexbox and Grid
- Font Awesome
